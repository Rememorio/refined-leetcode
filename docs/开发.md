# 开发

## 环境

- Node.js 22 或更高版本
- pnpm 11（由 `packageManager` 字段锁定）
- Chrome 或其他 Chromium 浏览器

```sh
git clone git@github.com:Rememorio/refined-leetcode.git
cd refined-leetcode
corepack enable
pnpm install
```

## 检查与构建

```sh
pnpm test
pnpm lint
pnpm build
```

生产构建位于 `dist/`。在 `chrome://extensions/` 开启开发者模式，选择“加载已解压的扩展程序”并指向该目录。

## 开发服务器

```sh
pnpm dev
```

前端代码支持热更新。`background` 代码改动后，需要在 Chrome 扩展管理页点击“重新加载”。

## 发布

`master` 使用 Conventional Commits 和 semantic-release：

- `fix:` 生成补丁版本。
- `feat:` 生成次版本。
- 破坏性变更生成主版本。

GitHub Actions 会依次运行测试、格式检查和生产构建，然后生成带 ZIP 产物的 GitHub Release。
